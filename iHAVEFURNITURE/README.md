โดยเราจะทำกันทั้งหมด 5 คนและมีหน้าที่แตกต่างกันไปในแต่ละตำแหน่ง
ซึ่งเราจะทำกันเกี่ยวกับ web application e-commerce โดยได้เรื่องบรรดาลใจมาก I Have CPU 
เราจึงเลือกทำเว็บนี้ขึ้นมาเป็นโปรเจคในรายวิชา
